    Section 5.4 First Proofs with Sets

    In the previous section we looked at many basic properties of sets. Now we will venture into some proofs that are more indicative of your future classes, and return (momentarily) back to the safety of numbers.
    We first examine how one would prove a subset to be true.

    Proving \(A\subseteq B\).

    Let \(x\in A\)
    ...
    Therefore \(x\in B\)
    The keen-eyed student will notice that this is simply the direct proof applied to the definition of subset, specifically:
    \begin{equation*} \forall x\in\mathcal{U} \text{ }x\in A\implies x\in B \end{equation*}
    We will see many examples of this proving method throughout out the class, yet before we jump in, I’d like to note for either the instructor or the diligent student who has been paying very close attention to the journey. I used to try and teach this by taking away the numbers, and instead using the \({\color{red}\text{beginning}}\text{,}\) \({\color{green}\text{muddle}}\) and the \({\color{blue}\text{end}}\) in Subsection 5.3.2 instead of the now off-handed way you see presented here, but I believe the students felt like it was too jarring and then leaned away from subsets and set proofs, which was detrimental to their learning once we got to abstract algebra. So now I introduce these proofs with numbers in hopes that you leave the set theory section with the love that you left from the even and odd proofs.

    Subsection 5.4.1 The First Proof

    Define the sets
    \begin{equation*} A=\{x\in\Z\text{ }|\text{ }x=6a\text{ for some } a\in\Z\} \end{equation*}
    \begin{equation*} B=\{x\in\Z\text{ }|\text{ }x=2a\text{ for some } a\in\Z\} \end{equation*}
    Prove: \(A\subseteq B\)
    Before we begin any proof we sing our song (play-along):
    "What’s the \({\color{red}{P}}\text{?}\)"
    Answer 1.
    \(x\in A\)
    "What’s the \({\color{blue}{Q}}\text{?}\)"
    Answer 2.
    \(x\in B\)
    "What’re the definitions?"
    Answer 3.
    The definition of the sets written directly above silly goose!
    "Now, what to do?"
    Answer 4.
    A direct proof! (proof of a subset!)

    Subsection \({\color{red}{\text{The Beginning}}}\)

    To begin this proof we follow our procedure above, by choosing an arbitrary element of the set \(A\text{,}\) using words in the manner of:
    \begin{equation*} \color{red}{\text{Let }s\in A} \end{equation*}
    Thats all there is to it, we abscond with a random element of \(A\text{,}\) then drudge forward with our journey to show it is a member of \(B\text{.}\)

    Subsection \({\color{green}{\text{The Muddle}}}\)

    I chose to begin with this example as \(\color{green}{\text{the muddle}}\) is almost identical to most other of our examples. That is we simply state what it takes for \(s\) to belong to \(A\text{.}\) From the definition above of the set \(A\text{,}\) we see in the condition part of the set-builder notation it tells us that there must be an integer so that \(s\) is 6 times this integer (sound familiar?). To invoke this we could write something like:
    By the definition of membership of \(A\text{,}\) we can produce some integer \(t\) so that \(s=6t\text{.}\)

    Subsection \({\color{blue}{\text{The End}}}\)

    Now, for the end we need to finally get to the finish line, that is finally conclude that our element \(s\) is indeed a member of \(B\text{.}\) For this notice that
    \begin{equation*} s=6t=2\cdot(3t) \end{equation*}
    and since 3 is an integer and \(t\) is an integer we have that \(3t\) is an integer and hence \(s\) satisfies the condition to be in \(B\text{.}\)
    Next, we present it all in one place.
    Example 5.4.1.
    Define the sets
    \begin{equation*} A=\{x\in\Z\text{ }|\text{ }x=6a\text{ for some } a\in\Z\} \end{equation*}
    \begin{equation*} B=\{x\in\Z\text{ }|\text{ }x=2a\text{ for some } a\in\Z\} \end{equation*}
    Prove: \(A\subseteq B\)
    Proof:
    Let \(s\in A\text{,}\) by definition of the set \(A\) we can find an integer \(t\) so that \(s=6t\text{,}\) yet note since \(6=2\cdot 3\) we have \(s=6t=2\cdot(3t)\text{.}\) Now, since \(3\) and \(t\) are integers we can conclude \(3t\) is an integer. Therefore by the definition of the set \(B\) we have that \(s\in B\)

    Subsection 5.4.2 More Examples

    This is an important concept of your future mathematics courses so we will now provide a couple more abbreviated examples of subset.

    Example 5.4.2.

    Consider the sets:
    \begin{equation*} A=\{x\in\Z\text{ }|\text{ }x=6a+4\text{ for some }a\in\Z\} \end{equation*}
    \begin{equation*} B=\{y\in\Z\text{ }|\text{ }y=3a+1\text{ for some }a\in\Z\} \end{equation*}
    Prove: \(A\subseteq B\)
    Proof.
    We begin by selecting a completely arbitrary element of \(A\text{,}\) which we will denote as \(n\in A\text{.}\) Applying the condition of membership to \(A\) we can find an integer \(m\) such that \(n=6m+4\text{.}\) By factoring we see that
    \begin{equation*} n=6m+4=6m+3+1=3(2m+1)+1 \end{equation*}
    and since \(2m+1\) is an integer we see that \(n\) satisfies the condition to be a member of \(B\text{.}\)
    Therefore, we have chosen an arbitrary \(n\in A\text{,}\) and successfully shown that \(n\in B\text{,}\) hence we have shown that \(A\subseteq B\text{.}\)
    For our next example we will look at proof involving the intersection (Definition 5.2.8).

    Example 5.4.3.

    Consider the sets:
    \begin{equation*} A=\{x\in\Z\text{ }|\text{ }x=2a+1\text{ for some }a\in\Z\} \end{equation*}
    \begin{equation*} B=\{y\in\Z\text{ }|\text{ }y=5a+2\text{ for some }a\in\Z\} \end{equation*}
    \begin{equation*} C=\{z\in\Z\text{ }|\text{ }z=10a+7\text{ for some }a\in\Z\} \end{equation*}
    Prove: \(C\subseteq A\cap B\)
    Proof.
    We first choose an arbitrary element of \(C\) in attempts to show it also belongs to \(A\cap B\text{,}\) name this element \(x\in C\text{.}\)
    Using the definition of the set \(C\) we see that we can fix a specific \(s\in\Z\) so that \(x=10s+7\text{.}\)
    Now, to show membership of the intersection, \(A\cap B\text{,}\) we need to show that our element \(x\) is both a member of \(A\) and that it is a member of \(B\text{,}\) by the definition of intersection (Definition 5.2.8). Thus we break the remainder of the proof into two parts.
    [Membership of \(\mathbf{A}\)]
    Notice that we can factor
    \begin{equation*} x=10s+7=10s+6+1=2(5s+3)+1 \end{equation*}
    and as \(5s+3\) is an integer we see that \(x\) satisfies the conditions to be a member of \(A\text{.}\)
    [Membership of \(\mathbf{B}\)]
    Similarly, notice that we can again factor
    \begin{equation*} x=10s+7=10s+5+2=5(2s+1)+2 \end{equation*}
    and as \(2s+1\) is an integer we see that \(x\) satisfies the conditions to be a member of \(B\text{.}\)
    Finally, since we have chosen a completely arbitrary element \(x\in C\) and then have shown that \(x\in A\) and that \(x\in B\) we can conclude that \(x\in A\cap B\text{,}\) therefore we have shown that \(C\subseteq A\cap B\text{.}\)
    Our next example will work through a proof that involves a union (Definition 5.2.5).

    Example 5.4.4.

    Consider the sets
    \begin{equation*} A=\{c\in\Z\text{ }|\text{ }c=6g+4\text{ for some }g\in\Z\} \end{equation*}
    \begin{equation*} B=\{d\in\Z\text{ }|\text{ }d=15h-8\text{ for some }h\in\Z\} \end{equation*}
    \begin{equation*} C=\{f\in\Z\text{ }|\text{ }f=3j+1\text{ for some }j\in\Z\} \end{equation*}
    Prove: \(A\cup B\subseteq C\)
    Proof.
    To start the proof we will choose an arbitrary element of \(A\cup B\text{,}\) we will name this random member \(y\in A\cup B\text{.}\) By the definition of union (Definition 5.2.5) we have that \(y\in A\) or \(y\in B\text{,}\) thus we split our proof in to two cases.
    [Case 1: \(\mathbf{y\in A}\)]
    For this first case, assume that \(y\in A\text{,}\) by the condition defining the set \(A\) we can find an integer \(t\) such that \(y=6t+4\text{.}\) Next, we calculate:
    \begin{equation*} y=6t+4=6t+3+1=3(2t+1)+1 \end{equation*}
    since \(2t+1\) is an integer we see that \(y\) satisfies the condition to be a member of \(C\text{,}\) that is \(y\in C\text{.}\)
    [Case 2: \(\mathbf{y\in B}\)]
    For this case we will instead assume \(y\in B\text{.}\) This time by the condition defining the set \(B\) we can locate a special integer \(s\) so that \(y=15s-8\text{.}\) Now we can calculate:
    \begin{equation*} y=15s-8=15s-9+1=3(5s-3)+1 \end{equation*}
    and since \(5s-3\) is an integer we see that, as the condition for membership of \(C\) is satisfied, \(y\in C\text{.}\)
    To wrap-up, since we have chosen a random element \(y\in A\cup B\) and have shown that this same element must have the property that \(y\in C\) we can conclude that \(A\cup B\subseteq C\text{.}\)
    Now we will explore the process of showing when sets are equal.

    Definition 5.4.5. Equals.

    We say that two sets \(A\) and \(B\) are equal, denoted
    \begin{equation*} A=B \end{equation*}
    if and only if
    \begin{equation*} A\subseteq B \text{ and } B\subseteq A \end{equation*}
    Therefore to prove the statement \(A=B\) we must have two proofs, one proof for \(A\subseteq B\) and one proof for \(B\subseteq A\text{.}\)
    Our next couple of examples aim to show this.

    Example 5.4.6.

    Consider the set
    \begin{equation*} X=\{x\in\Z\text{ }|\text{ }x=3g+2h\text{ for some }g,h\in\Z\} \end{equation*}
    Prove: \(X=\Z\)
    Proof.
    We will again need to break this proof into two parts.
    [Want to Show: \(\mathbf{X\subseteq\Z}\)]
    For the first part, let \(c\in X\) and hence by the definition of the set \(X\) we have that \(x\in\Z\text{,}\) therefore \(X\subseteq\Z\text{.}\) (this is of course what many refer to as the easy part)
    [Want to Show: \(\mathbf{\Z\subseteq X}\)]
    In this next part, assume \(m\in\Z\text{.}\) It is now our objective to show membership in \(X\text{,}\) to do such we will need to produce two integers, the way in which we discovered these integers is irrelevant to the proof, so for the sake of argument, lets just assume angels whispered it to me in my sleep.
    Notice that since \(m\) is an integer and \(-m\) is an integer, then we can calculate:
    \begin{equation*} 3m-2m=m \end{equation*}
    and hence by the condition of set \(X\) we have that \(m\in X\text{,}\) therefore \(\Z\subseteq X\text{.}\)
    To wrap-up, since we have shown that both \(X\subseteq \Z\) and \(\Z\subseteq X\) we may conclude that \(X=\Z\text{.}\)
