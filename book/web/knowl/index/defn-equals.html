Definition 5.2.4. Equal Sets.

We say that a set \(A\) is equal to a set \(B\text{,}\) denoted \(A= B\text{,}\) if and only if both
\begin{equation*} A\subseteq B \end{equation*}
and
\begin{equation*} B\subseteq A \end{equation*}
or equivalently by:
\begin{equation*} \forall x\in\mathcal{U}\text{ }x\in A \iff x\in B \end{equation*}
