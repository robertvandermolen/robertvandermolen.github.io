Definition 7.4.1. Partition.

Let \(A\) be a non-empty set. \(\mathscr{P}\) is a partition of \(A\) if \(\mathscr{P}\) is a set of subsets of \(A\) such that:
  1. If \(X\in \mathscr{P} \text{, then }X\neq \emptyset\)
  2. If \(X\in \mathscr{P}\) and \(Y \in \mathscr{P}\text{,}\) then \(X=Y\) or \(X\cap Y = \emptyset\)
  3. \(\displaystyle \displaystyle\bigcup_{X\in\mathscr{P}} X = A\)
