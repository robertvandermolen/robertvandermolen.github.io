Definition 7.6.1. Surjection.

A function \(f:A \rightarrow B\) is onto \(B\text{,}\) or is a surjection means
\begin{equation*} \text{Rng}(f)=B\text{.} \end{equation*}
When \(f\) is a surjection, we write
\begin{equation*} f:A \twoheadrightarrow B \end{equation*}
or
\begin{equation*} f: A \overset{\text{onto}}{\rightarrow} B\text{.} \end{equation*}
