It is now time to take care of those statments that were not propositons from Example 1.1.3, but have propably been bugging you this whole time like: \(x+2=4\text{,}\) which are known as an open statment or predicate. But, how on earth are we to do mathematics and these sentences not be in our lexicon? To handle these we need to quantify the \(x\text{,}\) by saying we can find a number so that \(x+2=4\text{.}\) Before quantifying the statement you didn’t even need to assume \(x\) was a number!
