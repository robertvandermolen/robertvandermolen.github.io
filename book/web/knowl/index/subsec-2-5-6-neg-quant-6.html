Example 1.5.9.

For these examples we will see multiple ways to write them.
  1. \(\sim (\forall n\in \mathbb{Z}\) \(\sqrt{n} \in \mathbb{Z})\equiv (\exists n\in \mathbb{Z}\) \(\sqrt{n} \not\in \mathbb{Z})\)
    • *NOT* (For all integers \(n\text{,}\) \(\sqrt{n}\) is an integer)
    • For some integers \(n\text{,}\) \(\sqrt{n}\) is not an integer.
  2. \(\sim (\exists n \in \mathbb{Z}\) \(n^2 =5)\equiv[\forall n \in \mathbb{Z}\) \(n^2 \neq 5]\)
    • *NOT*(There exists an integer \(n\) such that \(n^2 =5\))
    • For all integers \(n\text{,}\) \(n^2\) does not equal \(5\text{.}\)
