Definition 1.5.10. Unique Existential Quantifier.

For a predicate \(P(x)\text{,}\) the proposition
\begin{equation*} \exists! x\text{ }P(x) \end{equation*}
is read
\begin{equation*} \text{"there exists a unique }x\text{ such that }P(x)\text{"} \end{equation*}
and is true iff the truth set of \(P(x)\) has exactly one element form the universe. The symbol \(\exists!\) is called the unique existential quantifier.
