Definition 6.2.3. Principle of Mathematical Induction.

Let \(k\in\N\text{,}\) and let \(S\subseteq\N\) with the following properties:
  1. \(\displaystyle k\in S\)
  2. \(S\) is an inductive set
Then \(S\) contains all natural numbers greater than or equal to \(k\text{,}\) that is
\begin{equation*} S=\{n\in\N\text{ }|\text{ }n\geq k\} \end{equation*}
