Example 2.2.3.

For this example lets set a few propositons:
\begin{equation*} P:\text{ It is sunny today} \end{equation*}
\begin{equation*} Q:\text{ It is colder than yesterday} \end{equation*}
\begin{equation*} R:\text{ We will go hiking} \end{equation*}
\begin{equation*} S:\text{ We will go for a bike ride} \end{equation*}
\begin{equation*} W:\text{ We will be home by dark} \end{equation*}
Next consider the list of premises.
  • \(\displaystyle \sim P\wedge Q:\text{ It is not sunny today and it is colder than yesterday}\)
  • \(\displaystyle R\implies P:\text{ We will go hiking only if it is sunny}\)
  • \(\displaystyle (\sim R)\implies S:\text{ If we do not go swimming then we will go hiking}\)
  • \(\displaystyle S\implies W:\text{ If we go on a bike ride then we will be home before dark}\)
When can then make the following arguments:
  1. To start we will only consider the first premise.
    \((\sim P)\wedge Q\)
    \(\therefore \text{ }\sim P\)
    Since we have assumed that it is not sunny and colder than yesterday we can conclude it is not sunny outside, using conjunction.
  2. Now Let’s use the conclusion from (A), with the next premise.
    \((\sim P)\)
    \(R\implies P\)
    \(\therefore \text{ }\sim R\)
    That is if we assume that it is not sunny and that we will go hiking only if it is sunny, then we can conclude that we do not go hiking, using Modus Tollens.
  3. Now using the conclusion from (B) and our next premsise we see
    \((\sim R)\)
    \((\sim R)\implies S\)
    \(\therefore \text{ } S\)
    Since we have assumed that we are not hiking and that if we don’t hike then we will go on a bike ride we can thus conclude that we will go on a bike ride by Modus Ponens.
  4. Now using the conclusion of (C) and our next premise.
    \(S\)
    \(S\implies T\)
    \(\therefore \text{ }W\)
    Hence since we assumed that we are going on a bike ride and if we go on a bike ride we will be back home by dark we can conclude that we will be back by dark again by Modus Ponens.
We have just done our first proof, and shown if all the statements are assumed then we can conclude that we will be home before dark!
