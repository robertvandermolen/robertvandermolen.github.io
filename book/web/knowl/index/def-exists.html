Definition 1.5.1. Existential Quantifier.

For a predicate \(P(x)\text{,}\) the sentence
\begin{equation*} \exists x\text{ } P(x) \end{equation*}
is read
\begin{equation*} \text{"There exists an }x\text{ such that }P(x)\text{"} \end{equation*}
or
\begin{equation*} \text{"For some }x,\text{ }P(x)\text{"} \end{equation*}
or
\begin{equation*} \text{"We can find an }x\text{ so that }P(x)\text{"} \end{equation*}
The proposition \(\exists x \text{ }P(x)\) is true iff the truth set of \(P(x)\) is nonempty. The symbol \(\exists\) is called the existential quantifier
