Definition 7.5.1. Function.

A function, or mapping, from \(A\) to \(B\) is a relation \(f\) from \(A\) to \(B\) such that:
  1. The domain of \(f\) is \(A\text{.}\)
  2. If \((x,y)\in f\) and \((x,z)\in f\) then \(y=z\)
For a function \(f\) we write
\begin{equation*} f: A \rightarrow B \end{equation*}
We say
\begin{equation*} \text{"}f \text{ is a function from } A \text{ to } B\text{"} \end{equation*}
or
\begin{equation*} \text{"}f \text{ maps }A\text{ to }B\text{"} \end{equation*}
The set \(B\) is called the codomain of \(f\text{.}\)
In the case where \(B=A\text{,}\) we say that
\begin{equation*} f\text{ is a function on }A\text{"} \end{equation*}
When \((x,y)\in f\text{,}\) we write \(y=f(x)\text{.}\)
