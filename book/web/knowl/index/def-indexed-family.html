Definition 5.7.8. Indexed Family of Sets.

Let \(\Delta\) be a nonempty set such that for every \(i\in\Delta\) we correspond a set \(A_i\text{.}\) An indexed family of sets over \(\Delta\text{,}\) is the family of sets,
\begin{equation*} \mathscr{A}=\{A_i\text{ }|\text{ }i\in\Delta\} \end{equation*}
We call the set \(\Delta\) the indexing set.
