Definition 6.4.1. The Fibonacci Numbers.

(i) (Bases Cases)
\begin{equation*} f_1=1 \text{ and } f_2=1 \end{equation*}
(ii) (Inductive Case)
To calculate any Fibonacci number, \(f_n\text{,}\) we add the two numbers that preceded it:
\begin{equation*} f_n = f_{n-1} + f_{n-2} \end{equation*}
