Definition 1.5.7. Universal Quantifier.

For a predicate \(P(x)\text{,}\) the statement
\begin{equation*} \forall x\text{ }P(x) \end{equation*}
is read
\begin{equation*} \text{"For all }x\text{ }P(x)\text{"} \end{equation*}
and is true iff the truth set of \(P(x)\) is the entire universe. The symbol \(\forall\) is called the universal quantifier.
