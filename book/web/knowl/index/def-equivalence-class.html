Definition 7.3.19. Equivalence Class.

Let \(R\) be an equivalence relation on a set \(A\text{.}\) For \(x \in A\) the equivalence class of \(x\) modulo \(R\) is the set
\begin{equation*} \bar{x} := \{y\in A \text{ }|\text{ } xRy\} \end{equation*}
Each element of \(\bar{x}\) is called a representative of this class.
