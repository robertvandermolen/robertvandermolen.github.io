Definition 7.6.6. Injection.

A function \(f:A \rightarrow B\) is one-to-one, or is an injection, means
whenever \(f(x)=f(y)\) then \(x=y\text{.}\)
We write this as
\begin{equation*} f:A \hookrightarrow B\text{,} \end{equation*}
or
\begin{equation*} f: A \overset{\text{1-1}}{\rightarrow} B\text{.} \end{equation*}
