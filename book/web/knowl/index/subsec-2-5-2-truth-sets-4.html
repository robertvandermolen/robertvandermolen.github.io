Example 1.5.1.

The truth set for the open sentence "\(x^2\lt 3\)" depends on the universe of discourse. When the universe is set as \(\mathcal{U}=\mathbb{N}\) then the truth set is \(\{0,1\}\text{.}\) When the universe is \(\mathcal{U}=\mathbb{Z}\) then the truth set is \(\{-1,0,1\}\text{.}\) If the universe was to be \(\mathcal{U}=\mathbb{R}\) then the truth set would be the open interval \((-\sqrt{3},\sqrt{3})\text{.}\)
