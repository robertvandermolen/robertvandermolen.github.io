Modus Tollens.

Modus Tollens is simply the application of the contrapositive, specifically: \(P\implies Q\text{,}\) and \(\sim Q\text{,}\) and concludes \(\sim P\text{.}\) That is
\(P\implies Q\)
\(\sim Q\)
\(\therefore\text{ } \sim P\)
A similar example of this is "If it is raining then the ground is wet. The ground is not wet. Therefore it is not raining."
