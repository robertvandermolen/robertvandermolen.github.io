Definition 7.1.1. Relations.

Let \(A\) and \(B\) be sets. \(R\) is a relation from \(A\) to \(B\) if and only if \(R\) is a subset of \(A\times B\) That is,
\begin{equation*} R\subseteq A\times B \end{equation*}
When \(R\) is a relation from \(A\) to \(A\) we say it is a relation on \(A\text{.}\)
