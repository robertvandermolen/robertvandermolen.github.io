Definition 4.1.4. 1-d Integer Cone.

For any integers \(x_1\text{,}\) \(x_2\text{,}\) ..., \(x_n\) for some natural number \(n\) we say an integer \(y\) is in the 1-d integer cone formed by \(x_1\text{,}\) \(x_2\text{,}\) ..., \(x_n\) if and only if there exists integers \(a_1\text{,}\) \(a_2\text{,}\) ..., \(a_n\) so that
\begin{equation*} y=a_1x_1+a_2x_2+...+a_nx_n \end{equation*}
