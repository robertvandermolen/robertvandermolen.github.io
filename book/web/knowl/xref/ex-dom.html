Example 7.1.7.

In this example, consider the two sets
\begin{equation*} A=\{1,2,3,4,5\} \end{equation*}
and
\begin{equation*} B=\{a,b,c,d,e\} \end{equation*}
and define the relation \(R\) from \(A\) to \(B\) defined as
\begin{equation*} R=\{(1,b),(3,a),(4,d),(4,e)\} \end{equation*}
which can be visualized as:
For this relation we have the following
\begin{equation*} \text{Dom}(R)=\{1,3,4\} \end{equation*}
and
\begin{equation*} \text{Rng}(R)=\{a,b,d,e\} \end{equation*}
