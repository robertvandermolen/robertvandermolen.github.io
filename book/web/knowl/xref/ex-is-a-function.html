Example 7.5.4.

Consider the relation \(f\) from \(\Z\times\Z\) to \(\Z\) defined as
\begin{equation*} f=\{((a,b),c)\in\Z\times\Z\times\Z\text{ }|\text{ }c=3a+2b\} \end{equation*}
Prove: \(f\) is a function

Proof.

To prove that \(f\) is a function we need to show the two conditions of being a function.
Condition (i)
The first condition is an equality of sets, per usual we will split this into two parts
\(\mathbf{\text{Proof of }\text{Dom}(f)\subseteq\Z\times\Z}\)
Let \((a,b)\in\text{Dom}(f)\) by definition of \(f\) we have that \((a,b)\in\Z\times\Z\) as desired. (the room chants obviously!)
\(\mathbf{\text{Proof of }\text{Dom}(f)\supseteq\Z\times\Z}\)
Let \((x,y)\in\Z\times\Z\text{,}\) then \(((x,y),3x+2y)\in f\) by definition of \(f\text{,}\) as desired.
Since we have shown both \(\text{Dom}(f)\subseteq\Z\times\Z\) and \(\text{Dom}(f)\supseteq\Z\times\Z\) we can conclude \(\text{Dom}(f)=\Z\times\Z\text{.}\) Therefore we have shown that \(f\) satisfies condition (i) of being a function.
Condition (ii)
Assume \(((n,m),c)\in f\) and \(((n,m),d)\in f\text{,}\) hence by definition of \(f\) we have that \(c=3n+2m\) and that \(m=3n+2m\) hence \(c=d\) as desired.
Since we have shown both conditions to be a function we can conclude that \(f\) is a function.
