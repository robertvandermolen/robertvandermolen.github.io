    Section 7.3 Equivalence Relations

    Equivalence relations show up a lot in your future math courses especially in algebra and number theory. Yet, before we dive into an equivalence relation lets look at the three properties which make it up. First up is the reflexive property.

    Definition 7.3.1. Reflexive.

    We say that a relation \(R\) on a set \(A\) is reflexive on \(A\) whenever
    \begin{equation*} \forall x\in A,\text{ } xRx \end{equation*}
    For our first example in this section we will return to our cartoons.

    Example 7.3.2.

    Consider the following set \(A\)
    and consider the relation \(R\) on \(A\) defined as
    or as a set
    notice that for every element of \(A\) it is in an ordered pair with itself, and hence \(R\) is reflexive.
    Let see a couple of examples of proof involving reflexive.

    Example 7.3.3.

    Consider the relation \(S\) on \(\Z\) defined as
    \begin{equation*} S=\{(a,b)\in\Z\times\Z\text{ }|\text{ }3|(a+2b)\} \end{equation*}
    Prove: \(S\) is reflexive

    Proof.

    To prove this we let \(a\in\Z\text{,}\) to show \(a\) is included in an order pair with membership to \(S\) amounts to proving an existential.
    That is we will need to produce an integer to create an ordered pair with \(a\text{.}\) Well, that integer is \(a\text{.}\)
    To see this simply note that
    \begin{align*} a+2a \amp=3\cdot(a) \end{align*}
    and since \(a\) is an integer by definition of divides \(3\) divides \(a+2a\text{,}\) and hence \((a,a)\in S\text{.}\)
    Since we have chosen \(a\) arbitrary we can conclude that \(S\) is reflexive as desired.

    Example 7.3.4.

    Let \(A\) be a set, and let \(R\) be a relation on \(A\text{.}\)
    Prove: if \(R\) is reflexive then \(\text{Dom}(R)=A\)

    Proof.

    Assume \(R\) is reflexive on \(A\text{.}\) As usual to show equality of sets (dont forget relations are sets!) we will need to break our proof into two parts.
    \(\mathbf{\text{Proof of }\text{Dom}(R)\subseteq A}\)
    As usual, let \(a\in\text{Dom}(R)\text{,}\) hence by definition we can find a \(b\in A\) such that \((a,b)\in R\text{,}\) since \(R\) is a relation on \(A\) we have that \(a\in A\text{.}\)
    \(\mathbf{\text{Proof of }\text{Dom}(R)\supseteq A}\)
    This time, let \(x\in A\text{,}\) since we have assumed \(R\) is reflexive, by definition \((x,x)\in R\) and hence by definition of domain \(x\in\text{Dom}(R)\text{.}\)
    Since we have shown both \(\text{Dom}(R)\supseteq A\) and \(\text{Dom}(R)\subseteq A\) we have that \(\text{Dom}(R)= A\) as desired.
    The next property that defines an equivalence relation is symmetric.

    Definition 7.3.5. Symmetric.

    We say that a relation \(R\) on a set \(A\) is symmetric on \(A\) whenever
    \begin{equation*} \forall x,y\in A,\text{ if } xRy\text{ then } yRx \end{equation*}

    Example 7.3.6.

    Consider the set
    \begin{equation*} B=\{1,2,3,4,5,6\} \end{equation*}
    and the relation \(T\) on \(B\) defined as
    \begin{equation*} T=\{(1,2),(1,5),(6,5),(2,1),(5,1),(5,6)\} \end{equation*}
    This relation is symmetric to see this unlike with reflexive we need not check every element of \(B\) we instead check every element of \(T\)
    \begin{equation*} \text{Since }(1,2)\in T\text{ we need (and have) }(2,1)\in T \end{equation*}
    \begin{equation*} \text{Since }(1,5)\in T\text{ we need (and have) }(5,1)\in T \end{equation*}
    \begin{equation*} \text{Since }(6,5)\in T\text{ we need (and have) }(5,6)\in T \end{equation*}
    now we have exhausted all the elements of \(T\) and thus \(T\) is indeed symmetric.
    Lets explore an example of a proof involving the symmetric property next.

    Example 7.3.7.

    Consider the relation \(R\) on \(\Z\) defined as
    \begin{equation*} R=\{(a,b)\in\Z\times\Z\text{ }|\text{ }3|(a+b)\} \end{equation*}
    Prove: \(R\) is symmetric

    Proof.

    To prove that \(R\) is symmetric we must choose an arbitrary \((a,b)\in R\text{.}\) By definition of \(R\) \(3|(a+b)\text{,}\) by definition of divides we can find an integer \(k\) such that \(a+b=3k\) yet since \(a+b=b+a\) we have that \(b+a=3k\) and hence \(3|(b+a)\) and thus \((b,a)\in R\text{.}\) Hence \(R\) is symmetric.

    Example 7.3.8.

    Prove: If \(S\) is a symmetric relation on \(A\text{,}\) and \(R\subseteq S\text{,}\) then \(R^{-1} \subseteq S\)

    Proof.

    Let \(R\) be a relation on the set \(A\text{.}\) Assume \(S\) is a symmetric relation on \(A\) and \(R\subseteq S\text{.}\) Let \(x\in R^{-1}\text{.}\) Thus by definition of relation, and definition of inverse, we can find \((p,j)\in A\) such that \(x=(p,j)\text{.}\) Thus, by definition of inverse, \((j,p)\in R\text{.}\) Thus by our assumption and the definition of subset, \((j,p)\in S\text{.}\) And, since \(S\) is symmetric, \((p,j)\in S\text{.}\) Since \((p,j) = x\text{,}\) \(x\in S\)
    Since \(x\in R^{-1}\) and \(x\in S\text{,}\) by definition of subset, \(R^{-1} \subseteq S\)
    The final property which makes an equivalence relation is transitive.

    Definition 7.3.9. Transitive.

    We say that a relation \(R\) on a set \(A\) is transitive on \(A\) whenever
    \begin{equation*} \forall x, y, z \in A, \text{ if } xRy \text{ and } yRz, \text{ then } xRz \end{equation*}

    Example 7.3.10.

    Consider the relation \(T\) on \(\Z\) defined as
    \begin{equation*} T=\{(a,b)\in\Z\times\Z\text{ }|\text{ }6|(a-b)\} \end{equation*}
    Prove: \(T\) is transitive on \(\Z\)

    Proof.

    To prove that \(T\) is transitive we must choose two elements \((a,b)\in T\) and \((b,c)\in T\text{.}\)
    By definition of \(T\) both \(6|(a-b)\) and \(6|(b-c)\text{.}\) Hence by the definition of divides we can find \(s\in\Z\) and \(t\in\Z\) such that \(a-b=6s\) and \(b-c=6t\text{.}\) Calculate:
    \begin{align*} a-c=(a-b)+(b-c) \amp=6s+6t \\ \amp=6(s+t) \end{align*}
    since \(s+t\) is an integer we can conclude that \(6|(a-c)\) and hence \((a,c)\in T\text{.}\) Therefore by the definition of transitive \(T\) is transitive.

    Example 7.3.11.

    Prove: \(R\) is transitive if and only if \(R\circ R \subseteq R\)

    Proof.

    Let \(R\) be a relation on a non-empty set \(A\)
    \(\mathbf{\text{Proof of }R\text{ is transitive}\implies R\circ R\subseteq R}\)
    Assume \(R\) is transitive. Let \((x,y)\in R\circ R\text{.}\) By definition of composition, we can find \(z\in A\) such that \((x,z)\in R\) and \((z,y) \in R\text{.}\) Since \(R\) is transitive, \((x,y)\in R\text{.}\)
    Since \((x,y)\in R\circ R\) and \((x,y) \in R\text{,}\) by definition of subset, \(R\circ R \subset R\text{.}\)
    \(\mathbf{\text{Proof of }R\text{ is transitive}\impliedby R\circ R\subseteq R}\)
    Assume \(R\circ R \subseteq R\text{.}\) Let \((x,y) \in R\) and \((y,z)\in R\text{.}\) Since \((x,y)\in R\) and \((y,z)\in R\text{,}\) by definition of composition, \((x,z)\in R\circ R\text{.}\) Hence by our assumption that \(R\circ R \subseteq R\text{,}\) and definition of subset, \((x,z)\in R\text{.}\)
    Thus, by definition of transitive, \(R\) is transitive.

    Proposition 7.3.12.

    Let \(A\) be a non-empty set. For the power set \(\mathscr{P}(A)\text{,}\) the relation "is a subset of" is reflexive on \(\mathscr{P}(A)\text{,}\) and transitive, but not symmetric.

    Proof of Proposition7.3.12.

    [Reflexive]
    Let \(X\in \mathscr{P}(A)\text{.}\) (Need To Show: \(X\subset X\)) Assume \(a\in X\text{.}\) Hence, \(a\in X\text{,}\) thus by definition of subset, \(X\subset X\) and hence "subset of" is reflexive.
    [Transitive]
    Let \(X, Y, Z \in \mathscr{P}(A)\) such that \(X\subset Y\) and \(Y\subset Z\text{.}\) Let \(a\in X\text{.}\) Thus by definition of subset, since \(X\subset Y\text{,}\) \(a\in Y\text{.}\) Since \(Y\subset Z\text{,}\) and \(a\in Y\text{,}\) \(a\in Z\) Thus, by definition of subset, since \(a\in X\) and \(a\in Z\text{,}\) \(x\subset Z\text{.}\) Hence "subset of" is transitive.
    [NOT Symmetric]
    notice \(\emptyset \subset A\) but \(A \not\subset \emptyset\) as we assumed that \(A\) was not empty hence "subset of" is not symmetric.
    Now we have collected all of the properties which make an equivalence relation, the definition is almost self evident.

    Definition 7.3.13. Equivalence Relation.

    A relation \(R\) on a set \(A\) is an equivalence relation on \(A\) if \(R\) is reflexive, symmetric, and transitive on \(A\text{.}\)

    Example 7.3.14.

    Consider the relation \(E\) on \(\Z\) defined as
    \begin{equation*} E=\{(a,b)\in\Z\times\Z\text{ }|\text{ }5|(a-b)\} \end{equation*}
    Prove: \(E\) is an equivalence relation

    Proof.

    The definition of equivalence relation has three properties:
    • reflexive
    • symmetric
    • transitive
    thus a proof that a relation is an equivalence relation should be split into three parts.
    Proof of Reflexive:
    To see that \(E\) is reflexive we start with an arbitrary element \(a\in\Z\text{.}\) Now for this relation we can calculate:
    \begin{align*} a-a \amp=0 \\ \amp=5\cdot 0 \end{align*}
    since 0 is an integer \(a-a\) is thus divisible by \(5\text{,}\) and hence \((a,a)\in E\) as desired. Therefore \(E\) is reflexive.
    Proof of Symmetric:
    To show that \(E\) is symmetric we begin by choosing a pair \((b,c)\in\Z\times\Z\) such that \((b,c)\in E\text{.}\) By the definition of \(E\) \(5|(b-c)\) thus by definition of divides we can find an integer \(k\) so that \(b-c=5k\) and hence \(c-b=-5k\) and since \(-k\) is an integer by definition of divides 5 divides \(c-b\) thus \((c,b)\in E\text{;}\) therefore \(E\) is symmetric.
    Proof of Transitive:
    To prove \(E\) is transitive we begin with two arbitrary elements of \(E\text{,}\) namely \((d,e)\in E\) and \((e,f)\in E\text{.}\) By definition of \(E\) 5 divides both \(d-e\) and \(e-f\text{,}\) thus by definition of divides we can find integers \(r\) and \(s\) such that \(d-e=5r\) and \(e-f=5s\text{.}\) Now calculate:
    \begin{align*} d-f=(d-e)+(e-f) \amp =5r+5s \\ \amp=5(r+s) \end{align*}
    siince \(r+s\) is an integer we have that \(d-f\) is divisible by \(5\) and thus \((d,f)\in E\) and hence \(E\) is transitive.
    As we have shown that \(E\) is reflexive, symmetric and transitive we have have verified that \(E\) is an equivalence relation as desired!
    For the next example we will find the following definition and lemma helpful.

    Definition 7.3.15. Parity.

    We say that two integers have the same parity when either both are even or both are odd.

    Lemma 7.3.16.

    For any integers \(a\) and \(b\) if \(a+b\) is divisible by 2 then \(a\) and \(b\) have the same parity.

    Proof of Lemma7.3.16.

    We will approach this proof exhaustive, using Lemma4.2.2 to break us into cases.
    Case 1: \(\mathbf{a}\) is even and \(\mathbf{b}\) is even
    In this case we assume that we can find integers \(k\) and \(\ell\) so that \(a=2k\) and \(b=2\ell\text{.}\) Calculate
    \begin{align*} a+b \amp=2k+2\ell \\ \amp=2(k+\ell) \end{align*}
    since \(k+\ell\) is an integer we have that \(a+b\) is even, hence when \(a\) and \(b\) are both even that \(a+b\) is even.
    Case 2: \(\mathbf{a}\) is odd and \(\mathbf{b}\) is even
    In this case we assume that we can find integers \(s\) and \(t\) so that \(a=2s+1\) and \(b=2t\text{.}\) Calculate
    \begin{align*} a+b \amp= (2s+1)+2t\\ \amp=2(s+t)+1 \end{align*}
    since \(s+t\) is an integer we have that \(a+b\) is odd. Hence when \(a\) is odd and \(b\) is even then \(a+b\) is odd.
    Case 3: \(\mathbf{a}\) is even and \(\mathbf{b}\) is odd
    In this case we assume we can find integers \(r\) and \(v\) so that \(a=2r\) and \(b=2v+1\text{.}\) Calculate
    \begin{align*} a+b \amp=2r+(2v+1) \\ \amp=2(r+v)+1 \end{align*}
    since \(r+v\) is an integer we have that \(a+b\) is odd. Hence with \(a\) is even and \(b\) is odd \(a+b\) is odd
    Case 1: \(\mathbf{a}\) is odd and \(\mathbf{b}\) is odd
    In this final case we assume we can find integers \(n\) and \(m\) so that \(a=2n+1\) and \(b=2m+1\text{.}\) Calculate
    \begin{align*} a+b \amp= (2n+1)+(2m+1)\\ \amp=2n+2m+1+1=2n+2m+2 \\ \amp=2(n+m+1) \end{align*}
    since \(n+m+1\) is an integer we conclude that \(a+b\) is even. Therefore when \(a\) is odd and \(b\) is odd we have that \(a+b\) is even.
    Therefore we see that the only time that \(a+b\) is even is exactly when \(a\) and \(b\) have the same parity.

    Example 7.3.17.

    Consider the relation \(R\) on \(\Z\) defined as
    \begin{equation*} R=\{(a,b)\in\Z\times\Z\text{ }|\text{ }2|(a+b)\} \end{equation*}
    Prove: \(R\) is an equivalence relation

    Proof.

    As we are proving that \(R\) is an equivalence relation we should split it into three parts.
    Proof of Reflexive:
    To show reflexive we start with an arbitrary element \(a\in\Z\text{,}\) then calculate
    \begin{align*} a+a \amp=2a \end{align*}
    and since \(a\) is an integer we have that \(a+a\) divisible by 2 and hence \((a,a)\in R\) and thus \(R\) is reflexive.
    Proof of Symmetric:
    To prove that \(R\) is symmetric choose an arbitrary element \((x,y)\in R\text{,}\) thus by definition of \(R\) we have that \(2|(x+y)\) hence we can find an integer \(k\) such that \(x+y=2k\text{,}\) yet note that \(y+x=x+y=2k\) hence \(y+x\) is divisible by 2 and therefore \((y,x)\in R\) and hence \(R\) is symmetric.
    Proof of Transitive:
    To prove transitive we choose elements of \(R\text{,}\) namely \((b,c)\in R\) and \((c,d)\in R\text{.}\) By definition of membership to \(R\) we have that \(2|(b+c)\) and \(2|(c+d)\text{,}\) that is that both \(b+c\) and \(c+d\) are even.
    By Lemma7.3.16 and Lemma4.2.2 we can break the rest of our proof in to a couple of cases.
    Case 1: \(\mathbf{c}\) is even
    Assume that \(c\) is even, by Lemma7.3.16 since we have that both \(b+c\) is even and \(c+d\) is even we can conclude that both \(c\) and \(d\) are even. Hence by the definition of even we can find integers \(n\) and \(m\) such that \(c=2n\) and \(d=2m\text{.}\) Calculate
    \begin{align*} b+d \amp=2n+2m \\ \amp=2(n+m) \end{align*}
    since \(n+m\) is an integer we have that \(2|(b+d)\) thus \((b,d)\in R\) as desired.
    Case 1: \(\mathbf{c}\) is odd
    Assume that \(c\) is odd, by Lemma7.3.16 then both \(b\) and \(d\) are odd as we have that both \(b+c\) and \(c+d\) are even. By definition of even we can find two integers \(s\) and \(t\) such that \(b=2s+1\) and \(d=2t+1\text{.}\) Calculate
    \begin{align*} b+d \amp=(2s+1)+(2t+1) \\ \amp=2s+2t+2 \\ \amp=2(s+t+1) \end{align*}
    since \(s+t+1\) is an integer we have that \(2|(b+d)\) and thus \((b,d)\in R\) as desired.
    Hence in both cases we have that \(R\) is transitive.
    As we have shown that \(R\) is reflexive, symmetric and transitive we have have verified that \(R\) is an equivalence relation as desired!

    Example 7.3.18.

    Let \(A\) be a set and let both \(R\) and \(S\) be relations on \(A\text{.}\)
    Prove: if \(R\) and \(S\) are equivalence relations then \(R\cap S\) is an equivalence relation.

    Proof.

    Let \(A\) be a set and let both \(R\) and \(S\) be relations on \(A\) and assume both \(R\) and \(S\) are equivalence relations. To prove that \(R\cap S\) is an equivalence relation we should break it down into the three conditions.
    Proof of Reflexive:
    To prove reflexive, we begin with an arbitrary element \(a\in A\text{.}\) Since \(R\) and \(S\) are equivalence relations they are in particular reflexive we have that \((a,a)\in R\) and \((a,a)\in S\text{,}\) hence by definition of intersection \((a,a)\in R\cap S\) thus \(R\cap S\) is reflexive.
    Proof of Symmetric:
    To prove symmetric, we choose an element \((x,y)\in R\cap S\text{.}\) By definition of intersection both \((x,y)\in R\) and \((x,y)\in S\text{.}\) Since both \(R\) and \(S\) are equivalence relations in particular they are both symmetric and hence \((y,x)\in R\) and \((y,x)\in S\text{,}\) thus by the definition of intersection \((y,x)\in R\cap S\text{.}\) Therefore \(R\cap S\) is symmetric.
    Proof of Transitive:
    To prove transitive we start with elements \((r,s)\in R\cap S\) and \((s,t)\in R\cap S\text{,}\) by definition of intersection we have that \((r,s)\in R\text{,}\) \((r,s)\in S\text{,}\) \((s,t)\in R\) and \((s,t)\in S\text{.}\) Since both \(R\) and \(S\) are equivalence relations, in particular they are transitive we have that \((r,t)\in R\) and \((r,t)\in S\text{,}\) hence by definition of intersection \((r,t)\in R\cap S\text{.}\) Therefore \(R\cap S\) is transitive.
    Hence since we have shown that \(R\cap S\) is reflexive, symmetric, and transitive we have that \(R\cap S\) is an equivalence relation.
    An equivalence relation has a natural subdivision on the set it is on.

    Definition 7.3.19. Equivalence Class.

    Let \(R\) be an equivalence relation on a set \(A\text{.}\) For \(x \in A\) the equivalence class of \(x\) modulo \(R\) is the set
    \begin{equation*} \bar{x} := \{y\in A \text{ }|\text{ } xRy\} \end{equation*}
    Each element of \(\bar{x}\) is called a representative of this class.

    Definition 7.3.20. Modulo.

    The set
    \begin{equation*} A/R = \{\bar{x} \text{ }|\text{ } x\in A\} \end{equation*}
    of all equivalence classes is called \(A\) modulo \(R\) (\(A\) mod \(R\))

    Example 7.3.21.

    Consider the set
    \begin{equation*} A=\{0,1,2,3,4\} \end{equation*}
    and define the relation \(R\) on \(A\) as follows
    \begin{equation*} R=\{(a,b)\in A\times A\text{ }|\text{ }3|(a+2b)\} \end{equation*}
    We claim that this is an equivalence relation of \(A\text{,}\) yet since we just have a finite set lets just go through and find all the relations, namely
    \begin{equation*} R=\{(0,0),(1,1),(2,2),(3,3),(4,4),(0,3),(3,0),(1,4),(4,1)\} \end{equation*}
    We can then go through and check that all the properties of an equivalence relation, notice for example that the following are all members of \(R\) (0,0), (1,1), (2,2), (3,3), (4,4) and hence \(R\) is reflexive.
    To continue note the following are all members of \(R\)
    \begin{align*} (0,0) \amp\text{ and }(0,0) \\ (1,1) \amp\text{ and }(1,1) \\ (2,2) \amp\text{ and }(2,2) \\ (3,3) \amp\text{ and }(3,3) \\ (4,4) \amp\text{ and }(4,4) \\ (0,3) \amp\text{ and }(3,0) \\ (1,4) \amp\text{ and }(4,1) \end{align*}
    and hence \(R\) is symmetric. For transitive we check them all again that all of the following are members of \(R\)
    \begin{align*} (0,0)\text{, }(0,3) \amp\text{ and }(0,3) \\ (0,3)\text{, }(3,0) \amp\text{ and }(0,0) \\ (3,3)\text{, }(3,0) \amp\text{ and }(3,0) \\ (0,3)\text{, }(3,3) \amp\text{ and }(0,3) \\ (1,1)\text{, }(1,4) \amp\text{ and }(1,4) \\ (4,1)\text{, }(1,1) \amp\text{ and }(4,1) \\ (4,4)\text{, }(4,1) \amp\text{ and }(4,1) \\ (4,1)\text{, }(1,1) \amp\text{ and }(4,1) \\ (1,4)\text{, }(4,4) \amp\text{ and }(1,4) \end{align*}
    Now note the following equivalence classes:
    \begin{align*} \bar{0}=\bar{3} \amp=\{0,3\} \\ \bar{1}=\bar{4} \amp=\{1,4\} \\ \bar{2} \amp=\{2\} \end{align*}

    Proposition 7.3.22.

    Let \(R\) be an equivalence relation on a nonempty set \(A\text{.}\) For all \(x\) and \(y\) in \(A\text{,}\)
    1. \(x\in \bar{x}\) and \(\bar{x} \subseteq A\)
    2. \(xRy\) if and only if \(\bar{x} = \bar{y}\)
    3. \(x\) is not related to \(y\) if and only if \(\bar{x}\cap \bar{y} = \emptyset\)

    Proof of Proposition7.3.22.

    We prove part (a) and part (b), leaving part (c) as an exercise.
    Proof of part (a)
    \(\mathbf{\text{Proof of }x\in\bar{x}}\)
    Let \(x\in A\text{,}\) since \(R\) is an equivalence relation, in particular it is reflexive, thus \(xRx\text{,}\) hence by definition of \(\bar{x}\text{,}\) \(x\in \bar{x}\)
    \(\mathbf{\text{Proof of }\bar{x} \subseteq A}\)
    Let \(\ell\in \bar{x}\text{.}\) Thus by definition of \(\bar{x}\text{,}\) \(xR\ell\text{.}\) Hence, \((x,l)\in R\text{.}\) Since \(R\) is a relation on \(A\text{,}\) \(R\subseteq A\times A\text{.}\) Hence by definition of cross product, \(\ell\in A\text{.}\)
    Proof of (b) \(xRy\) if and only if \(\bar{x} = \bar{y}\)
    Let \(R\) be a relation on a non-empty set \(A\)
    \(\mathbf{\text{Proof of }xRy\implies\bar{x}=\bar{y}}\)
    Let \((x, y) \in R\text{,}\) as our objective is to prove equality of sets we should break it into two parts.
    \(\mathbf{\text{Proof of }\bar{x}\subseteq\bar{y}}\)
    Let \(a \in \bar{x}\text{.}\) Thus by definition of equivalence class \(xRa\text{.}\) Since \(R\) is an equivalence relation, it is symmetric, and as we assumed \((x,y)\in R\text{,}\) we can thus conclude \((y,x) \in R\text{.}\) Since \(R\) is an equivalence relation, it is also transitive. And since we already concluded \((y,x)\in R\) and \((x,a) \in R\) we can now conclude \((y,a)\in R\text{.}\)
    Thus by definition of equivalence class, \(a\in \bar{y}\text{.}\) Thus by definition of subset, \(\bar{x} \subseteq \bar{y}\text{.}\)
    \(\mathbf{\text{Proof of }\bar{x}\supseteq\bar{y}}\)
    let \(a\in \bar{y}\text{.}\) Thus by definition of equivalence class \(yRa\text{.}\) Since \(R\) is an equivalence relation, it is transitive. And since we already concluded \((x,y)\in R\) and \((y,a) \in R\text{,}\) we can now conclude \((x,a) \in R\text{.}\)
    Thus by definition of equivalence class, \(a\in \bar{y}\text{.}\) Thus by definition of subset, \(\bar{x} = \bar{y}\)
    \(\mathbf{\text{Proof of }xRy\impliedby\bar{x}=\bar{y}}\)
    Let \(x \in A\) and \(y\in A\) such that \(\bar{x} = \bar{y}\text{.}\) Since \(R\) is an equivalence relation, it is reflexive. Hence, \(xRx\text{.}\) Thus by definition of \(\bar{x}\text{,}\) \(x\in \bar{x}\text{.}\) Since we assumed \(\bar{x} = \bar{y}\text{,}\) in particular \(\bar{x} \subseteq \bar{y}\text{,}\) \(x\in \bar{y}\text{.}\) By definition of \(\bar{y}\text{,}\) \(yRx\text{.}\) Since \(R\) is an equivalence relation, it is symmetric. Since we concluded \((y,x)\in R\text{,}\) we can conclude \((x,y)\in R\)
